## Requirements

Before starting üèÅ, you need to have [Git](https://git-scm.com), [Node](https://nodejs.org/en/) and [Yarn](https://yarnpkg.com) installed.

## Starting

```bash
# Clone this project
$ git clone https://github.com/Thiago-spart/tractian-challenger

# Access
$ cd tractian-challenger

# Install dependencies
$ yarn

# Run the project in developer mode
$ yarn dev

# The server will initialize in the <http://localhost:3000>
```
